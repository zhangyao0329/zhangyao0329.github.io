Start Ollama

无法使用systemd，我们可以直接运行Ollama：

# 后台运行
nohup /usr/bin/ollama serve > /tmp/ollama.log 2>&1 &

# 查看进程
ps aux | grep ollama

# 查看日志
tail -f /tmp/ollama.log

这会启动Ollama服务，但会占用当前终端。如果要后台运行：

# 后台运行
nohup /usr/bin/ollama serve > /tmp/ollama.log 2>&1 &

# 查看进程
ps aux | grep ollama

# 查看日志
tail -f /tmp/ollama.log

使用WSL的替代服务管理(推荐)

# 创建启动脚本
sudo vi /etc/init.d/ollama

在文件中添加：

#!/bin/bash
case "$1" in
    start)
        nohup /usr/bin/ollama serve > /tmp/ollama.log 2>&1 &
        ;;
    stop)
        pkill -f "ollama serve"
        ;;
    status)
        pgrep -f "ollama serve" > /dev/null && echo "Running" || echo "Stopped"
        ;;
    *)
        echo "Usage: $0 {start|stop|status}"
        exit 1
        ;;
esac

然后：

# 给脚本执行权限
sudo chmod +x /etc/init.d/ollama

# 使用方式
sudo /etc/init.d/ollama start

sudo /etc/init.d/ollama status

sudo /etc/init.d/ollama stop

Adding Ollama as a startup service (推荐)

（wsl1版本无法使用，会有下图bug）:

Create a user and group for Ollama:

sudo useradd -r -s /bin/false -U -m -d /usr/share/ollama ollama
sudo usermod -a -G ollama $(whoami)

Create a service file in /etc/systemd/system/ollama.service:

sudo vi /etc/systemd/system/ollama.service
[Unit]
Description=Ollama Service
After=network-online.target

[Service]
ExecStart=/usr/bin/ollama serve
User=ollama
Group=ollama
Restart=always
RestartSec=3
Environment="PATH=$PATH"

[Install]
WantedBy=multi-user.target

Then start the service:

main.py

import ollama


def main():
    client = ollama.Client()

    try:
        # 列出可用模型
        models = client.list()
        print("可用模型:")
        for model in models.models:
            print(f"  - {model.model} (参数大小: {model.details.parameter_size})")

        # 使用第一个模型进行对话
        if models.models:
            model_name = models.models[0].model
            print(f"\n使用模型: {model_name}")

            # 进行对话
            response = client.generate(
                model=model_name,
                prompt="Hello, how are you? Please respond briefly."
            )
            print(f"\nAI 响应: {response.response}")

            # 如果有思考过程，也显示出来
            if hasattr(response, 'thinking') and response.thinking:
                print(f"\n思考过程: {response.thinking}")

    except Exception as e:
        print(f"操作出错: {e}")


if __name__ == "__main__":
    main()

ollama_test.py

方法名 方法介绍 示例
Client() 创建Ollama客户端实例 client = ollama.Client()
client.list() 列出所有可用的模型 models = client.list()
client.generate() 使用指定模型生成文本响应 response = client.generate(model='name', prompt='text')
client.show() 显示指定模型的详细信息 model_info = client.show('model-name')
client.ps() 显示当前正在运行的模型 running_models = client.ps()
client.pull() 从仓库拉取模型 client.pull('model-name')
client.push() 推送模型到仓库 client.push('model-name')
client.create() 创建自定义模型 client.create(model='name', modelfile='content')
client.delete() 删除指定模型 client.delete('model-name')
方法 返回对象/属性 说明
client.list() ListResponse 包含模型列表的响应对象
models.models 模型对象列表
model.model 模型名称
model.details.parameter_size 模型参数量
client.generate() GenerateResponse 生成响应的结果对象
response.response AI生成的文本内容
response.thinking 思考过程（如果模型支持）
client.show() ModelInfo 模型的详细信息对象
client.ps() List[Process] 正在运行的模型进程列表

demo2:

import streamlit as st
import ollama
import time

# 初始化Ollama客户端
client = ollama.Client()

# 设置页面标题和图标
st.set_page_config(
    page_title="AI聊天助手",
    page_icon="🤖",
    layout="wide"
)

# 标题
st.title("🤖 AI聊天助手")
st.divider()

# 初始化聊天历史
if "messages" not in st.session_state:
    st.session_state.messages = []

# 显示聊天历史
for message in st.session_state.messages:
    with st.chat_message(message["role"]):
        st.markdown(message["content"])

# 用户输入
if prompt := st.chat_input("请输入您的问题..."):
    # 添加用户消息到历史
    st.session_state.messages.append({"role": "user", "content": prompt})

    # 显示用户消息
    with st.chat_message("user"):
        st.markdown(prompt)

    # 显示AI回复（带加载效果）
    with st.chat_message("assistant"):
        message_placeholder = st.empty()

        try:
            # 调用Ollama生成回复
            response = client.generate(
                model='deepseek-r1:1.5b',
                prompt=prompt,
                stream=False
            )

            # 获取回复内容
            full_response = response['response']

            # 模拟流式输出效果
            simulated_response = ""
            for chunk in full_response.split():
                simulated_response += chunk + " "
                message_placeholder.markdown(simulated_response + "▌")
                time.sleep(0.05)

            message_placeholder.markdown(simulated_response)

            # 添加AI回复到历史
            st.session_state.messages.append({"role": "assistant", "content": full_response})

        except Exception as e:
            error_message = f"抱歉，出现错误：{str(e)}"
            message_placeholder.markdown(error_message)
            st.session_state.messages.append({"role": "assistant", "content": error_message})

# 侧边栏 - 清空聊天记录
with st.sidebar:
    st.header("设置")

    if st.button("清空聊天记录"):
        st.session_state.messages = []
        st.rerun()

    # 显示当前模型信息
    st.divider()
    st.subheader("模型信息")
    try:
        models = client.list()
        if models.models:
            current_model = models.models[0]
            st.write(f"**模型名称**: {current_model.model}")
            st.write(f"**参数量**: {current_model.details.parameter_size}")
            st.write(f"**量化级别**: {current_model.details.quantization_level}")
    except:
        st.write("无法获取模型信息")

# 页脚
st.divider()
st.caption("基于Ollama和Streamlit构建的AI聊天助手")
